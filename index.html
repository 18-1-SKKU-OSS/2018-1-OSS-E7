프로젝트 소개 : bCNC

bCNC는 기존의 CNC 밀링 머신을 제어하던 GBRL 프로그램에서 좀 더 자동화 되고 편리한 편집기능을 제공하는 프로그램 입니다.

Translation

bCNC는 다양한 언어 패키지를 지원합니다. 여러 국가에서 상용화 될 정도로 규모가 있는 프로그램인 만큼 러시아어, 중국어, 독일어등 다양한 언어가 내장되어 있지만, 한국어는 아직까지 번역이 존재하지 않습니다. 저희는 한국인들의 접근성을 높이기 위해 설치부터 주요 절차의 번역을 진행하였습니다.

Code Edit

bCNC는 기능적으로는 다소 전문적인 기술의 프로그램이기 때문에 수치의 조정보다는, GUI기반의 프로그램인 만큼 UI/UX의 불편함을 개선하거나, 상세한 주석 및 주요소스 파악을 중점으로 두고 있습니다.

Explanation

설치부터 사용방법에 이르기까지 기존의 사이트는 많은 내용을 전부 나열하는 식으로 보여주고 있습니다. 저희는 상세한 과정에 대한 설명과 프로그램에 대한 소개를 진행합니다.

1. 간단한 클램프

도입

내 수제 클램프는 비록 나를 잘 대우 했음에도 불구하고 대체품이 필요했습니다. MDF 30x100mm의 직사각형 블록으로 만들어졌으며 한쪽면에 "둥근"MDF 조각이 붙어 있습니다. 가장 큰 어려움은 재료의 얇은 부분 (예 : 3mm의 MDF)을 고정하는 것이 었습니다. 클램프의 높이가 고정 되었기 때문에 클램프가 너무 많이 구부러져 작업 물이 빠져 나갔습니다. (사진) T-nuts와 역 스크류를 사용하여 재료의 높이를 조정하는 새로운 클램프 세트를 만드는 것이 아이디어였습니다

디자인

나는 librecad로 아주 빨리 뭔가를 스케치했다. 원형 가장자리가있는 30x100mm 직사각형으로 70mm x 5.2mm의 중간에 슬릿이 있고 6.5mm 구멍이 있습니다. 슬릿은 상단 나사를 자유롭게 움직여 올바른 위치로 조정하기위한 것입니다. 나는 M5 나사를 사용하고 있으므로 나사가 자유롭게 움직일 수있는 0.2mm의 공차를 허용합니다. 클램프를 제작 한 후에는 조금 더 상처를 입지 않을 것이라고 말하고 싶습니다. 예 : 허용 오차는 0.4mm입니다. (사진) 6.5mm의 구멍은 거꾸로 된 나사 (M5)의 너트를 삽입하는 것입니다.

gcode 만들기

1) bCNC 프로그램을 시작하고 메뉴에서 또는 직접 명령 줄에서 dxf 파일을 엽니다. $ bCNC MyClamps.dxf 2) Tools 탭을 클릭하고 사용할 EndMill (Bit)을 선택하십시오. 처음에는 아무 것도 없으므로 [+]를 클릭하고 새 것을 추가하십시오. 가장 중요한 매개 변수는 직경 bCNC입니다. (사진) 3) 재료를 선택하고 적절하게 수정하십시오. 1 플루트 엔드 밀이있는 MDF 재료의 경우 DOC 1.5mm z 단계에서 1200mm / min을 사용합니다. 4) Stock 오브젝트를 선택하고 두께를 19.5mm 두께의 MDF로 설정하십시오.

프로파일 만들기

도구 탭으로 이동 프로필 도구 선택 외부 직사각형을 선택하십시오. 적절한 엔드 밀 설정 바깥 쪽을 선택하십시오. 화면 하단의 실행을 클릭하십시오. (사진) 또는 명령 줄에서 명령 프로필을 입력하기 만하면됩니다. 생성 될 새 경로와 회색으로 표시 될 원본이 표시됩니다. 두 개의 내부 내부 경로에서도 동일한 작업을 수행하십시오. 둘 다 선택하고 입력하려면 [Control]을 클릭하십시오. 잘라 내기 도구 선택 모든 프로필 경로 선택 실행을 클릭하십시오 (또는 명령 컷을 입력하십시오) ISO1보기를 선택하면 다음 이미지와 같은 경로가 절단되어야합니다. (사진) 중요 사항 : 기계로 보내기 전에 먼저 내부 부품을 자르기위한 경로를 다시 정렬해야합니다. 이렇게하려면 편집기 탭으로 이동하여 편집기에서 이동할 경로를 선택하고 Ctrl 키 또는 Control 키를 사용하여 위 / 아래로 이동하십시오.

밀링

스톡 피스를 작업대 표면에 놓습니다. 원점 복귀주기 실행 수동으로 갠트리를 원하는 0 위치 (키를 사용하여) 또는 마우스 (Ctrl + Shift + 클릭)로 이동하거나 캔버스에 *** g *** 앤 트리 키를 입력하고 원하는 위치에서 마우스로 클릭하십시오. X-Y, Y-Z, X-Z 플롯에서만 작동합니다. WCS 탭에서 0 위치를 설정하거나 명령 행에서 set 0 0 0을 입력하십시오. X 축 또는 세트에 0을 입력하여 축을 하나씩 보낼 수 있습니다. Y 축 등 0 저는 일반적으로 종이를 사용하여 Z 방향으로 0을 찾고 용지에 닿을 때까지 갠트리를 0.1mm 간격으로 움직입니다 (더 이상 용지를 움직일 수 없음). 그런 다음 Z 좌표로 0.1mm를 설정합니다. . 0.1, 내가 사용하고있는 종이의 전형적인 두께가 0.1mm이기 때문에. 갠트리를 조금 들어 올린다. 3-5mm 실행을 클릭하십시오. Editor

GCode는 모든 CNC 기계의 모국어이며, 편집기를 사용하여 수정할 수 있으며 변경 사항이 무엇인지 확인할 수 있습니다. 예를 들어 보겠습니다. 가공 프로그램을로드 한 후 편집기 탭으로 이동하십시오. 여기서 우리는 멋진 개 프로필을 가지고 있으며, 왼쪽의 CAM은 bCNC에 의해 다른 섹션으로 부드럽게 그룹화 된 Gcode를 생성합니다. 이 예에서는 HEADER와 두 개의 BLOCKS를 볼 수 있습니다. 이들 각각에는 제목에 체크 상자가있어서 필요하지 않은 경우 특정 지역을 비활성화 할 수 있습니다. 편집 -> 블록 삽입 [Ctrl-B] 메뉴를 사용하여 더 많은 사용자 정의 블록을 추가 할 수 있습니다.Select Select 도구 [S]를 사용하여 개 머리를 선택합시다 : 선택한 Gcode 프로그램이 텍스트 편집기에서도 강조 표시됩니다. 너무 멋지다, 그치? 이제 Pan Move Object [M]를 사용하여 선택 영역을 마우스로 드래그합니다. 우리 개는 이제 기린처럼 보입니다! 그리고 너무 쉬웠습니다. 이것은 빙산의 일각에 불과합니다. 몇 가지 특수 명령을 시도하고 상단 메뉴에서 도구 -> 대칭 -> 수평으로 이동하십시오. (또는 명령 줄을 사용하십시오 : MIRROR HORIZONTAL (또는 MIR H 친구)결과는 외부 CAM 소프트웨어를 사용하지 않고도 몇 가지 공통 작업을 얼마나 간단하게 수행 할 수 있는지 보여줍니다.라인을 선택하고 아래 그림과 같이 편집 -> 라인 삽입 [Ctrl-Enter]을 사용하여 Gcode 프로그램에 새로운 라인 int를 삽입 할 수 있습니다 : 새 줄을 편집하고 기본 창에서 결과를 즐기십시오. 편집기는 실행 취소 [Ctrl-Z] 및 다시 실행 [Ctrl-Y]을 지원하므로 Gcode 파일을 작성하고 수정 해보십시오. 새 파일에 저장하거나 원래 파일을 덮어 씁니다.

Command Line

키보드보다 마우스를 좋아하는 사람들은 커맨드 라인입니다! Space 또는 Ctrl + Space를 눌러 어디에서나 명령 행으로 이동할 수 있습니다. 명령 행은 MDI (예 : G1 X ## Y ## F ##)와 몇 가지 유용한 명령을 받아들입니다. 예를 들어 ABOUT은 "about"bCNC에 대한 정보를 보여줍니다. 아래의 문서에서 별표는 ABO 또는 ABOU를 입력하여 bCNC가 올바른 명령을 이해할 수 있음을 의미합니다. 일부 명령은 공백으로 구분하여 제공해야하는 인수를 허용합니다. 문서에서 매개 변수는 [대괄호] 안에 있습니다. 모든 명령은 대소 문자를 구분하지 않습니다. ABO * UT : 서프라이즈! about 상자를 보여줍니다. ABS * OLUTE : 절대 좌표 (G90)를 설정합니다. CLE * AR : 터미널 지우기 BOX [dx] [dy] [dz] [nx] [ny] [nz] [tool] : 손가락 상자를 만드십시오. (이것은 쉽지 않습니다. CONT * ROL [TabName] : 지정된 탭으로 전환합니다. CUT [높이] [패스 당 깊이] : 선택된 블록을 절단 높이로 복제하고, 기본값은 활성 재료에서 가져옵니다. DOWN : 선택한 블록을 커팅 순서대로 아래로 이동 UP : 선택한 블록을 커팅 순서대로 위쪽으로 이동 DRI * LL [깊이] [펙] : 모든 관통 점에서 천공 수행 FIL * TER : 텍스트가있는 필터 편집기 블록 ED * ITOR : 편집기 탭으로 전환합니다. HOME : 원점 복귀주기 수행 HOLE : 구멍을 만드십시오. IM * PORT : 커서 위치 # 또는 파일 끝에 gcode 또는 dxf가있는 파일 이름 가져 오기 INK * SCAPE : inkscape 결과로 인한 불필요한 Z 모션을 제거합니다. ISO1 : ISO1 투영으로 전환 ISO2 : ISO2 투영으로 전환 ISO3 : ISO3 프로젝션으로 전환 LO * AD [파일 이름] : g 코드가 들어있는 파일 이름로드 MIRROR [HORIZONTAL / VERTICAL *] : 선택한 객체를 수평 또는 수직으로 반사합니다. MOVE [| CENTER | BL | TL | TL | TR | UP | DOWN | x] y [z] : 선택한 객체를 마우스 또는 좌표로 이동합니다. ORI * GIN x y z : 모든 것을 -x -y -z로 이동하여 원점을 x, y, z로 이동하십시오. OPEN : Grbl에 대한 직렬 연결 열기 CLOSE : Grbl과의 직렬 연결 닫기 EX * IT : 프로그램 종료 일시 중지 : 일시 중지주기 PROF * ILE [offset] : 프로파일 경로 작성 REL * ATIVE : 상대 좌표로 전환 (G91) RESET : Grbl의 소프트 리셋 수행 REV * ERSE : 역방향 경로 실행 :로드 된 G 코드 실행 ROT * ATE [CCW | CW | FLIP | ang] [x0 [y0]] : 선택한 블록을 시계 반대 방향 (90) / 시계 방향 (-90) / 뒤집기 (180) 90deg 또는 특정 각도와 피벗 점 ROU * ND [n] : 모든 자릿수를 n 자릿수로 반올림합니다. RU * LER : 마우스 눈금자로 거리 측정 SAFE [z] : 안전 Z 이동 SA * VE [파일 이름] : 파일 이름 또는 기본 이름으로 저장 SET [x [y [z]]] : x, y, z 좌표를 현재 작업 공간으로 설정합니다. STEP [s] : 모션 스텝 크기를 s로 설정합니다. SPI * NDLE [ON | OFF | 속도] : 스핀들 켜기 / 끄기 STOP : 현재 실행 중지 TERM * INAL : 터미널 탭으로 전환 공구 [직경] : 절삭 공구 직경 설정 도구 : 도구 탭으로 변경하십시오. UNL * OCK : 잠금 해제 WCS [n] : 작업 공간 인덱스 n으로 전환합니다. XY : XY보기로 전환 YX : XY보기로 전환 XZ : XZ보기로 전환합니다. ZX : XZ보기로 전환 YZ : YZ보기로 전환합니다. ZY : YZ보기로 전환

팬던트 웹 리모트 컨트롤

펜던트는 백그라운드에서 자동으로 실행되는 간단한 웹 페이지로 구성되어있어 컴퓨터를 쉽게 원격 제어 할 수있는 bCNC의 유용한 기능입니다.

액세스하는 방법 웹 브라우저를 http : // IP : 8080 링크로여십시오. 여기서 IP는 bCNC가 실행중인 IP 주소입니다. bCNC가 실행중인 동일한 시스템에서 테스트를 원하면 http : // localhost : 8080을 사용할 수 있습니다. 케이블 / 무선 네트워크 pc / 테이블에서이 기능을 사용하는 경우 bCNC가 실행중인 IP 주소를 알아야합니다. 대개 4 개의 숫자 (예 : 192.168.0.100)이지만 실제 숫자는 여러 요소를 기반으로 변경됩니다. Windows OS에서 그들을 찾으려면 ipconfig 명령을 사용할 수 있습니다. unix에서는 ifconfig를 사용할 수 있습니다 (하지만 이미 알고 있어야합니다). 외부 네트워크에서 펜던트에 액세스하려면 포트를 통해 외부 요청을 bCNC로 전달하여 라우터를 구성해야합니다. 이것은 부적절한 방식으로 관리되고이 위키의 범위를 벗어나는 경우 안전하지 않습니다. 펜던트 사용하기 페이지가로드 된 후 일반적인 인터페이스와 동일한 기능을하는 익숙한 버튼이 표시됩니다. bCNC가 작동중인 컴퓨터에서 멀리 떨어진 곳에서 편안하게 작업 진행을 즐기십시오. 펜던트를 사용하여 bCNC를 실행하는 호스트 PC에 파일을 업로드 할 수도 있습니다. 파일 선택 대화 상자가 열리는 "파일 선택"버튼이 있습니다. 이렇게하면 로컬 파일 시스템을 탐색 할 수 있으며 확인되면 원격 PC의 사용자 폴더에 파일을 업로드합니다. bCNC가 아직 실행 중이 아닌 경우 새 파일이 자동으로로드되어 시작할 준비가됩니다. 공유 폴더를 구성하거나 usb 키를 사용하지 않고도 캠 스테이션에서 bCNC로 파일을 쉽게 이동할 수 있습니다. ### 카메라 라이브 스트림 http : // IP : 8080 / camera.html에 액세스하면 opencv 의존성을 설치 한 경우에도 카메라의 원격 비디오를 즐길 수 있습니다. 구성 구성 파일을 사용하여 펜던트 기능 실행을 중지하거나 다른 수신 대기 포트를 지정할 수 있습니다. 후드 펜더드는 bCNC가 만든 로컬 웹 서버 덕분에 작동합니다. 또한 몇 가지 특별한 요청을받습니다. / 명령 줄에서 입력되는 것으로 cmd? xxx를 보냅니다. / send? gcode = xxx는 기계에 직접 gcode를 보내고, % 0D가있는 여러 줄을 보냅니다. / 상태는 기계의 위치와 상태를 json으로 반환합니다. / config는 스핀들의 rpmmax를 반환합니다. / icon? name icons 폴더의 모든 icon.gif 파일 제공 / index.html을 반환합니다.
